# Aaroha Event Management System

A Django-based web application designed to manage and promote a charity event for underprivileged children.

---

##  Features

-  Event Details Page (Banner, Description, Date, Venue)
- Registration Form with Validation
-  My Events Page (View registrations)
- Responsive UI using Bootstrap
-  Dark Mode Support
-  Animations and Hover Effects

---

## 🛠️ Tech Stack

- Backend: Django (Python)
- Frontend: HTML, CSS, Bootstrap
- Database: SQLite
